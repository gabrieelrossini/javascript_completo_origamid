# Origamid - JavaScript Completo

Repositório com exercícios do curso **JavaScript Completo** da [Origamid](https://www.origamid.com/curso/javascript-completo-es6/).

## Módulos Concluídos

| Módulo | Pasta | Conteúdo |
|--------|-------|----------|
| Variáveis | `variaveis/` | `var`, `let`, `const`, escopo |
| Tipos de Dados | `tipos-de-dados/` | `string`, `number`, `boolean`, `null`, `undefined`, `object` |
| Números e Operadores | `numeros-e-operadores/` | Operadores aritméticos, `Math` |
| Boolean e Condicionais | `boolean-e-condicionais/` | `if/else`, operadores lógicos, ternário |

## Como Executar

1. **Navegador (recomendado):** Abra o `index.html` de cada pasta no navegador → Console (F12)
2. **VS Code:** Extensão "Live Server" → "Open with Live Server"
3. **Node.js:** `node variaveis/script-aula1.1.js`
